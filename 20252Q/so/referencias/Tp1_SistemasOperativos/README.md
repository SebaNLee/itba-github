# Tp1_SistemasOperativos

Primer Trabajo Práctico Especial de la materia Sistemas Operativos (72.11) del ITBA

## Objetivo General del Trabajo

Entender y desarrollar un sistema de procesos que se comuniquen y sincronicen en un sistema POSIX.

## Instrucciones de Compilación y Ejecución

+ Inicializar contenedor de docker:
    ```$> ./dockerRun.sh```
+ Moverse al root, Compilar y Linkeditar el proyecto:
    ```cd root```
    ```make```
+ Ejecutar el binario principal con los parámetros deseados:
    ```$> ./bin/master [parámetros]```
