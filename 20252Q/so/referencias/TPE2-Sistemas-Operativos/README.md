# TPE2-Sistemas-Operativos

Trabajo Práctico Especial número 2 de la materia Sistemas Operativos (72.11) del ITBA

## Descripción

Sistema operativo básico con kernel, scheduler, memory manager, semáforos y shell.

## Compilación

```bash
./compAndRun
```

## Comandos

- `help` - Lista de comandos
- `ps` - Ver procesos
- `kill <pid>` - Terminar proceso
- `echo <texto>` - Imprimir texto
- `cat` - Leer entrada
- `phylo` - Filósofos
- `test -h` - Tests disponibles
- `clear` - Limpiar pantalla
- `exit` - Salir

## Pipes

```bash
echo "test" | wc -w
cat | filter
```
